# Electric Vehicle (EV) Market Data Analysis ⚡🚗

Exploratory Data Analysis (EDA) dashboard examining Electric Vehicle growth indices, sales distributions, and infrastructure densities.

## 📌 Project Overview
As the transport sector shifts to sustainable electricity, understanding adoption patterns is vital for urban planners and charging grid developers. This repository analyzes regional EV registration records to expose registration hot spots, market growth rates, and manufacturer dominance curves.

## 🛠️ Tech Stack
- **Analysis:** Python, Pandas, NumPy
- **Visualizations:** Matplotlib, Seaborn
- **Documentation:** Markdown
- **Frontend Representation:** HTML, CSS (for web resume details)

## ⚙️ Analysis Insights & Deliverables
- **Adoption Curves:** Analyzed sales velocity to identify growth rates of Battery Electric Vehicles (BEVs) vs Plug-in Hybrid EVs (PHEVs).
- **Spatial Densities:** Grouped locations to detail charging-grid demand points based on car registrations.
- **Manufacturer Trends:** Highlighted dominant EV manufacturers and their market share curves.
- **Presentation:** Created structured clean dashboards and generated insights summaries for product presentations.
